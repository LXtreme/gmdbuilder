# Contributing

If you have ideas or features you would like to have added:
 - Open an Issue
 - Open a Pull Request
 - Contact me on Discord

## First contributions

There are many easy opportunities for contributions, especially for the [wrapper classes](https://lxtreme.github.io/gmdbuilder/wrappers.html). These wrapper classes could have more helper methods added to it, and many triggers are still missing wrappers.

The wrapper classes are found in the `src/gmdbuilder/classes` directory.
